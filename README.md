# ⚛️ React Project Setup

## 🚀 Getting Started

### 1️⃣ Clone the repository

```
git clone https://github.com/tanvirahmedrume/car-doctor.git
cd your-repo-name
```

---

### 2️⃣ Install dependencies

```
npm install
```

---

### 3️⃣ Run the project

```
npm run dev
```

---

## 🔐 Environment Variables

Create a `.env` file in the root directory and add:

```
VITE_API_URL=your_api_url
VITE_FIREBASE_API_KEY=your_key
```

---

## 📦 Build for production

```
npm run build
```

---

## 📁 Tech Used

* React
* Vite
* Firebase (if used)

---

## ⚠️ Notes

* Don't upload `.env` file to GitHub
* Use `.env.example` for reference

---

⭐ That's it! Your project should be running on `http://localhost:5173`
